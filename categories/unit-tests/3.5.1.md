# 3.5.1 - Do Not Ship Fixtures

Priority | ID | Category | Criteria | Impacts
---------|----|----------|----------|--------
Minor | [3.5.1][3.5.1] | [Unit Tests][unit-tests] | Put fixtures into a folder called `_fixtures`. | [Project Maintenance][PROJECT-MAINTENANCE]

Table of Contents:
- [Description](#description)
- [Discussion](#discussion)

## Description

Put fixtures into a folder called `_fixtures`.

_Fixtures_ are data types, data, and anything else that you define once, and then share amongst multiple unit tests. They only exist to help with writing unit tests. They are not production code.

This coding standard provides advice on how to meet [1.4.1].
